# STS에서 Jquery 사용해보기 

## 직접 다운받아서 사용하기 
- https://jquery.com/
- Download the compressed ~ 
- 크롬일 경우
  - 도구 더보기 
  - 페이지를 다른 이름으로 저장
- resources/static/js 폴더에 넣어준다 
- <head> 내부에 <script type="text/javascript" src="/js/jquery-3.6.0.min.js" ></script> 추가 
- 개인별로 경로 설정 해줘야 한다.
  
## CDN 호스트 사용
